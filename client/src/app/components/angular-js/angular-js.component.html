Angular Framework de desenvolvimento web

O Angular é um framework (ou plataforma) de desenvolvimento web criado pelo Google e usado para construir aplicativos web robustos e dinâmicos.

Utiliza TypeScript como linguagem principal e oferece uma arquitetura baseada em componentes, onde os aplicativos são divididos em componentes reutilizáveis.

Angular Plataforma de desenvolvimento web

Trata-se de um framework de desenvolvimento web de código aberto cujas principais características são:

Aplicação de página única Single-Page Application - SPA

Consiste em uma aplicação baseada em JavaScript que interage com o usuário através de uma página única que é carregada somente uma vez.

Os demais recursos da aplicação são carregados dinamicamente e adicionados à página em resposta às ações do usuário.

 Framework gratuito e de código aberto Desenvolvido e mantido pela Google

O framework Angular é desenvolvido e mantido principalmente pela companhia Google, porém conta com a colaboração da comunidade e outras companhias.
